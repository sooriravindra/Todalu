Todalu (Kannada for Lisp), is my attempt to write an interpreter for my own dialect of Lisp.

** Tentative features

|------------------------------+---------|
| Expression                   |  Result |
|------------------------------+---------|
| 1                            |       1 |
| (+ 1 1)                      |       2 |
| (* (/ 10 2) 3)               |      15 |
| ((lambda (x) (* x 2)) 10)    |      20 |
| (define id (lambda (x) (x))) |       - |
| (id 10)                      |      10 |
| (quote (+ 1 2)               | (+ 1 2) |
| (car (quote (1 2 3)))        |       1 |
| (cdr (quote (1 2 3)))        |   (2 3) |
| (define var 19)              |       - |
| var                          |      19 |
| (if (- 1 1) 3 10)            |      10 |
| (eq? var 19)                 |       1 |
| (> 10 12)                    |       0 |
| (< 8 19)                     |       1 |
| (not 12)                     |       0 |
|------------------------------+---------|
